# Challenge

Backend implemented using FastAPI, Python3 and sqlite3

    How to start backend

- cd backend
- Configure DB_PAYABLE_PATH inside .env file

- Install dependencies for Python using:

  python3 -r requirements.txt

  or

  python -r requirements.txt

- Run create_db.py located at backend/db

      py create_db.py

- Start backend by running main.py script

      py main.py

- Test REST API visiting FastAPI Documentation: http://localhost:5001/docs
